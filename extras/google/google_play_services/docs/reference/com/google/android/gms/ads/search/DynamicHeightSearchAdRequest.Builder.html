public DynamicHeightSearchAdRequest.Builder setAdTest (boolean adTest)

Indicates if the CSA "adTest" parameter is enabled. The adTest parameter is used to indicate that a request for ads is a test. When the adTest parameter has a value of on, Google treats the request as a test and does not count the ad impressions or track the clickthrough results.

public DynamicHeightSearchAdRequest.Builder setAdjustableLineHeight (int adjustableLineHeight)

The CSA "adjustableLineHeight" parameter. Specifies the line height for the title, description, and visible url ad elements. Note that the line height includes the ad element text. If the line height is greater than the text, the text is centered vertically and the remaining pixels are distributed equally as padding above and below the text.
